
# Web Sitesi Kurulum Rehberi

Bu dosyalar, GitHub Pages üzerinde ücretsiz olarak yayınlayabileceğiniz kişisel özgeçmiş web sitesini içerir.

## Adımlar

1. https://github.com adresine gidip ücretsiz bir hesap oluşturun.
2. Yeni bir repository oluşturun. İsmi tam olarak şu olmalı: `furkanyildiz.github.io`
3. Bu klasör içindeki tüm dosyaları GitHub reposuna yükleyin.
4. GitHub Pages otomatik olarak aktif olacak. Site adresiniz: https://furkanyildiz.github.io

İyi şanslar!
